# 📊 Amazon Product Reviews - Data Visualization Project

## 📌 Project Overview

This project focuses on transforming raw Amazon product review data into meaningful visualizations that support business decision-making. Using Python libraries such as Pandas, Matplotlib, and Seaborn, multiple charts and a dashboard were created to identify customer behavior, product trends, and review patterns.

---

## 🎯 Project Objectives

- Transform raw data into meaningful visualizations.
- Explore customer review patterns.
- Build attractive charts using Python.
- Present business insights through visual storytelling.
- Develop a professional portfolio project.

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

- Most customers gave 4-star and 5-star ratings.
- Most customers recommended the products.
- Amazon products dominate the review count.
- Negative reviews are generally longer.
- Review patterns help understand customer satisfaction.

## ▶️ How to Run the Project

1. Clone this repository.

```bash
git clone <repository-url>
```

2. Navigate to the project folder.

```bash
cd CodeAlpha_Data_Visualization_Project
```

3. Install the required libraries.

```bash
pip install -r requirements.txt
```

4. Open the notebook.

```bash
jupyter notebook
```

5. Run all cells in **Data_Visualization.ipynb**.
